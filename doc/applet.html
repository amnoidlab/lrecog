Java Applet

If you have a Java capable internet browser a small Java Swing based button should load on this page and you should be able to click on this button to load and start the LeavesRecognition application as a java applet.
If an error occurs please make sure that you have at least Java-Plug-In v1.3.1 installed.

Please note that loading and starting of the applet may take some time accordingly to the network bandwith that is currently available and the number of predefined images. So please be patient until it is fully initialized.

The loading & starting process may take up to 2-5 minutes!!!!
